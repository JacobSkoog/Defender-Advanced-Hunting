# Defender Advanced Hunting

KQL queries for Microsoft Defender XDR Advanced Hunting, focused on detection engineering across identity, endpoint, and cloud app telemetry.

Queries are mapped to [MITRE ATT&CK](https://attack.mitre.org/) where applicable and include inline documentation explaining the detection logic.

## Usage

Each `.md` file describes the detection scenario and contains the full KQL query. Copy the query into the Advanced Hunting blade in the [Microsoft Defender portal](https://security.microsoft.com) and adjust environment-specific variables (marked with comments) before running.
